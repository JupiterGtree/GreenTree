# Green Tree Document Changelog

## Version 2.0.0 - 2026-07-14

This version replaces the former staged-presale and locked-balance launch model with:

- a freely transferable token from launch;
- public DEX-based pricing;
- website acquisition at the same market-derived price;
- no buyer lock, vesting, staged unlock, or claim period;
- no custom Green Tree program required for Phase 1;
- cumulative liquidity targets of up to 18%, 22%, and 32% at the USD 50,000, USD 100,000, and USD 200,000 thresholds;
- a commitment to more than ten substantial, verifiable environmental missions per year;
- continued 2-of-2 Squads treasury control;
- unchanged fixed maximum supply and top-level allocation totals.
